## Global CSS — Autofill Fix

```css
input:-webkit-autofill,
input:-webkit-autofill:hover,
input:-webkit-autofill:focus,
input:-webkit-autofill:active {
  -webkit-box-shadow: 0 0 0 30px var(--input) inset !important;
  color: var(--text-primary) !important;
}
```

## Props — TextInput

| Prop | Type | Description |
| --- | --- | --- |
| `label` | `string` | Label above the input |
| `error` | `string` | Error message below |
| `value` | `string` | Controlled value |
| `setValue` | `ReactDispatch<string>` | State setter shorthand |
| `startIcon` | `ReactNode` | Icon on the left |
| `endIcon` | `ReactNode` | Icon on the right |
| `containerClassName` | `string` | Wrapper class |

Also accepts all native `<input>` props.

## Props — PasswordInput

Same as TextInput minus `endIcon`. Adds:

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `showToggle` | `boolean` | `true` | Show/hide visibility toggle |
