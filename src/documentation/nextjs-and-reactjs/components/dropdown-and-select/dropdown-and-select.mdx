# Dropdown And Select

Two menu primitives built on Radix UI with ripple feedback, label/error support, and consistent zinc styling.

## Dropdown

A trigger-agnostic dropdown menu — pass any element as `children` to act as the trigger.

<Demo
  source={DropdownSrc}
  code={`<Dropdown
  items={[
    { value: 'profile', label: 'Profile' },
    { value: 'settings', label: 'Settings' },
    { value: 'logout', label: 'Logout' },
  ]}
>
  <Button variant="secondary">
    Options <ChevronDown className="size-4" />
  </Button>
</Dropdown>`}
>
  <Dropdown

      { value: 'profile', label: 'Profile' },

## Props — Dropdown

| Prop | Type | Description |
| --- | --- | --- |
| `children` | `ReactNode` | Trigger element |
| `items` | `DropdownItem[]` | Menu items |
| `label` | `string` | Label above trigger |
| `error` | `string` | Error message below |
| `open` / `onOpenChange` | `boolean / fn` | Controlled open state |

## Props — Select

| Prop | Type | Description |
| --- | --- | --- |
| `options` | `SelectOption[]` | List of options |
| `placeholder` | `string` | Placeholder text |
| `startIcon` / `endIcon` | `ReactNode` | Icons inside trigger |
| `setValue` | `ReactDispatch<string>` | State setter shorthand |
| `label` / `error` | `string` | Label and error text |
