## Components

| Component | Element | Description |
| --- | --- | --- |
| `Table` | `<table>` | Scrollable wrapper + table |
| `TableHeader` | `<thead>` | Header section with zinc-800 bg |
| `TableBody` | `<tbody>` | Body, removes last row border |
| `TableRow` | `<tr>` | Row with bottom border |
| `TableHead` | `<th>` | Header cell |
| `TableCell` | `<td>` | Data cell |

All components accept `className` and forward native HTML props.
