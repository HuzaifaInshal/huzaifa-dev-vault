# Modals And Sheet

Dialog-based overlays built on Radix UI `Dialog`. Both require a `title` prop (rendered as `Dialog.Title` for accessibility — visually hidden via `sr-only`).

## Modal

export const ModalDemo = () => {
  const [open, setOpen] = useState(false)
  return (
    <>
      <Button onClick={() => setOpen(true)}>Open Modal</Button>
      <Modal open={open} onOpenChange={setOpen} title="Example Modal" className="w-full max-w-md p-6">
        <div className="flex items-start justify-between mb-4">
          <h2 className="text-lg font-semibold text-zinc-100">Modal Title</h2>
          <ModalClose />
        </div>
        <p className="text-sm text-zinc-400 mb-6">
          This is the modal body. Place any content here.
        </p>
        <div className="flex justify-end gap-2">
          <Button variant="ghost" onClick={() => setOpen(false)}>Cancel</Button>
          <Button onClick={() => setOpen(false)}>Confirm</Button>
        </div>
      </Modal>
    </>
  )
}

## Props — Modal

| Prop | Type | Description |
| --- | --- | --- |
| `open` | `boolean` | Controlled open state |
| `onOpenChange` | `(open: boolean) => void` | Open state handler |
| `title` | `string` | Accessible title (sr-only) |
| `children` | `ReactNode` | Modal content |
| `className` | `string` | Content panel class |

## Props — Sheet

Same as Modal, plus:

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `side` | `'left' \| 'right'` | `'right'` | Slide-in direction |
