# Figma to JSON

A Python script that calls the Figma API and returns a pruned, AI-friendly JSON representation of any Figma node. Feed the output directly into the [UI from Figma JSON](/frontend/nextjs/prompts/ui-from-figma-json) prompts.

## Installation

Save the script somewhere on your machine (e.g. `~/scripts/figma_extractor.py`), then make it globally accessible:

```bash
chmod +x ~/scripts/figma_extractor.py
sudo ln -s ~/scripts/figma_extractor.py /usr/local/bin/generate_figma_json
```

Set your Figma access token as an environment variable (add to `~/.bashrc` or `~/.zshrc`):

```bash
export FIGMA_ACCESS_TOKEN="your_token_here"
```

## Usage

```bash
generate_figma_json <figma_url>
```

Pass any Figma design URL that contains a `node-id` parameter. The script prints the pruned JSON to stdout.

## What the pruning keeps

The script strips most Figma metadata and retains only what's useful for code generation:

| Property | Why it's kept |
|----------|--------------|
| `id`, `name`, `type` | Node identity |
| `visible` | Visibility toggles |
| `children` | Tree structure |
| `absoluteBoundingBox` | Position and size |
| `fills`, `strokes`, `strokeWeight` | Visual styles |
| `characters`, `style` | Text content and typography |
| `layoutMode`, `primaryAxisAlignItems`, `counterAxisAlignItems` | Flex/auto layout direction and alignment |
| `paddingLeft/Right/Top/Bottom`, `itemSpacing` | Spacing |
| `cornerRadius` | Border radius |
