# formatDate

Two tiny date helpers built on the native `Intl.DateTimeFormat` API — no dependencies.

- **`formatDate`** — formats a date as a readable string (`April 21, 2026`)
- **`formatRelative`** — formats a date as a relative time string (`3h ago`, `just now`)

## Usage

```ts
import { formatDate, formatRelative } from '@/utils/formatDate'

formatDate(new Date())
// → 'April 21, 2026'

formatDate('2025-01-01', { month: 'short' })
// → 'Jan 1, 2025'

formatRelative(Date.now() - 1000 * 60 * 5)
// → '5m ago'

formatRelative(new Date('2026-04-14'))
// → 'April 14, 2026'  (older than 7 days → falls back to formatDate)
```
